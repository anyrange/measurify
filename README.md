## Development

### Setup environemnt

```bash
$ git clone https://github.com/anyrange/spotify-worm.git
$ cd spotify-worm
$ npm install
$ npm run dev
```

### Environment variables

Rename [.env.example](/.env.example) to `.env` and don't forget to set your variables
